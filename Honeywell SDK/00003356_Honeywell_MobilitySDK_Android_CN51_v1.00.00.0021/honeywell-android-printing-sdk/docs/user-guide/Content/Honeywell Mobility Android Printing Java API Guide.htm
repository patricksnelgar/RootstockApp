Honeywell Mobility Android Printing Java API Guide

The Honeywell Mobility Android Printing Java API allows applications to print from a Honeywell Android computer to a Honeywell receipt or label printer.

Honeywell Printing Java API Reference

Printer Commands and Attributes User Guide

The Java classes support these features:

  • Communication with the printer via Bluetooth
  • Sending data strings or byte arrays to the printer for receipt and label printing or custom printer commands
  • Changing text font styles for receipt printing
  • Print bar codes and bitmap images
  • Providing the progress of the printing via progress events

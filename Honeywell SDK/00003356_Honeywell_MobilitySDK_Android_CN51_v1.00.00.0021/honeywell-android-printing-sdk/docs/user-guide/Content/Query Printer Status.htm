Query Printer Status

You may check the printer status before printing. If the returned status indicate one or more error conditions, you may prompt the user to correct the errors. The LabelPrinter and LinePrinter classes inherit the getStatus method from its parent class.

public int[] getStatus() throws PrinterException

To use this method, you must define the query command to send to the printer and the patterns to search in the printer response. Each pattern is associated with a message number. When a matching pattern is found, its message number is returned in the integer array. The returned value of the getStatus method is an array because it may hold multiple message numbers if multiple errors were detected.

The standard printer_profiles.JSON file provided by the Honeywell Mobility Android Printing SDK contains predefined entries for ESC/P status queries. Please refer to the Command and Attribute Format > Printer Status Commands and Attributes section of the Printer Commands and Attributes User Guide for more information.
